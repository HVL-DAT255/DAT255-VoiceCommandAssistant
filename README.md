## Voice-Activated Tic-Tac-Toe

This project is a voice-controlled Tic-Tac-Toe game powered by a machine learning model and deployed using Streamlit.

### Live Demo

Try the hosted version on Hugging Face Spaces:

https://mariushorn-voice-tictactoe.hf.space

---

## Run Locally

To test the application locally, run:

```bash
streamlit run app.py
```
